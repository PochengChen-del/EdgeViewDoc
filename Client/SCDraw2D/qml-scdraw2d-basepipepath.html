    管線用單一路徑繪製元件，支援直線與三次貝茲曲線。. More...

    Detailed Description

    BasePipePath 依據 pointList 所定義的節點資料， 自動產生可繪製的路徑座標。

    當相鄰節點包含有效控制點，且控制點不在直線段上時， 系統會自動轉換為三次貝茲曲線，以達到平滑管線效果。

    Note: 此元件屬於繪圖系統的低階組件， 通常作為 Pipe、BasePipe、BasePipeOverlap 的內部組成， 不建議直接用於一般 UI。

    Property Documentation

    fillGradient : var

    填色漸層設定。

    position 範圍為 0.0 ~ 1.0， 系統會自動補齊起點與終點的漸層色。


    gradientRadialRad : real

    放射性漸層旋轉角度（弧度）。

    此屬性由 radialGradientRad 自動換算， 主要供內部繪製使用。


    gradientStop : var

    填色漸層用的 Stop 陣列（position, color）。

    會自動轉換成內部渲染所需格式。


    lineGradient : var

    線條漸層設定。

    position 範圍為 0.0 ~ 1.0， 系統會自動補齊起點與終點的漸層色。


    lineGradientStop : var

    線條漸層用的 Stop 陣列（position, color）。

    會自動轉換成內部渲染所需格式。


    pointList : var

    路徑節點列表。

    每個節點至少需包含 x 與 y， 並可選擇性定義前後控制點，用於產生貝茲曲線。

    {
    x: Number,
    y: Number,
    preCtrl:  { x: Number, y: Number },
    nextCtrl: { x: Number, y: Number }
}

    radialGradientRad : real

    放射性漸層旋轉角度（單位：度）。

    可設定範圍為 -360 ~ 360， 正值為順時針旋轉，負值為逆時針旋轉。
