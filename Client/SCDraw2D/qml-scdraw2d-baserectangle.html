    Detailed Description

    BaseRectangle 是基於 ScRect 的自訂 QML 元件， 用於繪製矩形或圓角矩形，並支援：

    • 線性漸層描邊
    • 線性漸層填色
    • 放射性漸層（可旋轉）

    元件會依據 width、height 與 cornerRadius 自動計算繪製所需的 points 資料。

    Property Documentation

    cornerRadius : real

    矩形圓角半徑。

    當 cornerRadius 大於 0 時， 矩形四角會以圓角方式繪製。


    fillGradient : var

    填色使用的線性漸層設定。


    fillGradientCenterX : real

    放射性漸層中心點 X 座標。


    fillGradientCenterY : real

    放射性漸層中心點 Y 座標。


    fillGradientEndX : real

    線性填色漸層終點 X 座標。


    fillGradientEndY : real

    線性填色漸層終點 Y 座標。


    fillGradientRadiusX : real

    放射性漸層 X 軸半徑。


    fillGradientRadiusY : real

    放射性漸層中心點 Y 座標。


    fillGradientStartX : real

    線性填色漸層起點 X 座標。


    fillGradientStartY : real

    線性填色漸層起點 Y 座標。


    lineGradient : var

    描邊使用的線性漸層設定。

    陣列元素需包含：

    • position：0.0 ~ 1.0
    • color：顏色值

    radialGradientRad : real

    放射性漸層旋轉角度（單位：度）。

    有效範圍為 -360 至 360， 正值為順時針方向。
