    Detailed Description

    BasePipe 用於繪製由多段路徑構成的管線圖形， 支援直線與三次貝茲曲線，自動判斷控制點是否偏離線段。

    主要功能包含：

    • 多路徑（paths）支援
    • 虛線流動（Dash Flow）
    • 路徑方向控制
    • 線條與填充漸層

    Method Documentation

    _transPoint(prePt, curPt)

    管線的路徑集合，每一條路徑由點陣列組成。

    系統會根據 pathList 自動轉換為直線或貝茲曲線。
