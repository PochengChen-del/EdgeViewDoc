    Detailed Description

    ScPie 可以用於各種儀表板、進度條或動畫效果。 支援線性與放射性漸層填色，並可自動計算弧形上的點。

    範例用法

    ScPie {
    width: 200
    height: 200
    fillColor: "red"
    lineColor: "blue"
    fillRatio: 0.75
    gradientType: 6
    gradientRadialInfo: Qt.rect(50, 50, 40, 40)
}

    Property Documentation

    beClose : bool

    是否閉合形狀。


    beFillColor : bool

    是否填充顏色。


    capType : int

    線條末端樣式：0 (Butt) 或 1 (Round)。


    dashArray : list<real>

    虛線樣式列表 [線長, 間隔, ...]。


    dashOffset : real

    虛線偏移量。


    dashedImageMirror : bool

    是否鏡像顯示虛線貼圖。


    dashedImageSrc : string

    虛線處欲顯示的貼圖路徑。


    fillColor : color

    填充顏色。


    fillDirection : int

    填充方向：

    • 0: bottom to top
    • 1: top to bottom
    • 2: left to right
    • 3: right to left
    • 4: clockwise
    • 5: counterclockwise

    fillRatio : real

    填充比例，範圍 0.0~1.0。


    fillStartRad : real

    填充起始角度 (弧度 0~2\pi)。


    gradientAnyDir : rect

    當 gradientType 為 5 時，指定兩點方向的矩形區域。


    gradientRadialInfo : rect

    當 gradientType 為 6 時，指定徑向漸層資訊 (cx, cy, major_radius, minor_radius)。


    gradientRadialRad : real

    徑向漸層的旋轉角度或半徑參數。


    gradientStop : variant

    漸層停點，格式 [progress, color, ...]。 例如：[0.0, "#FF0000", 1.0, "#0000FF"]


    gradientType : int

    漸層類型：

    • 0: disable
    • 1: left-top to right-bottom
    • 2: left-bottom to right-top
    • 3: top to bottom
    • 4: left to right
    • 5: Any two points (使用 gradientAnyDir)
    • 6: Radial (使用 gradientRadialInfo)
    • 7: Angular

    joinType : int

    線條連接處樣式：0 (Bevel), 1 (Round), 或 2 (Miter)。


    lineColor : color

    線條顏色。


    lineGradientStop : variant

    線條漸層停點，格式 [progress, color, ...]。


    lineGradientType : int

    線條漸層類型：0: disable, 1: top-to-down, 2: left-to-right。


    lineWidth : real

    線條寬度。


    pieInfo : rect

    扇形資訊，包含中心點座標與起訖角度。


    pipeBackColor : color

    管線背景顏色。


    pipeFrontColor : color

    管線前端顏色（僅在形狀為管線時生效）。


    points : list<real>

    多邊形或弧形的點列表。


    rectRadius : real

    橢圓組件通常不使用此屬性（保留給矩形組件）。


    useDash : bool

    是否使用虛線。


    visiblePipeFrontColor : bool

    是否顯示管線前景顏色。
