    Detailed Description

    Property Documentation

    endAngle : real

    弧形結束角度（度數）。


    fillGradient : var

    填充漸層色陣列。每個元素包含 position(0~1) 與 color。


    fillGradientCenterX : real

    放射性漸層中心 X（0~1）。


    fillGradientCenterY : real

    放射性漸層中心 Y（0~1）。


    fillGradientEndX : real

    線性漸層終點 X（0~1）。


    fillGradientEndY : real

    線性漸層終點 Y（0~1）。


    fillGradientRadiusX : real

    放射性漸層半徑 X（0~1）。


    fillGradientRadiusY : real

    放射性漸層半徑 Y（0~1）。


    fillGradientStartX : real

    線性漸層起點 X（0~1）。


    fillGradientStartY : real

    線性漸層起點 Y（0~1）。


    gradientRadialRad : real

    放射性漸層旋轉角度（弧度）。

    由 radialGradientRad 轉換而來，會正規化到 -360~360 內再轉弧度。


    lineGradient : var

    弧線漸層色陣列。每個元素包含 position(0~1) 與 color。


    radialGradientRad : real

    放射性漸層旋轉角度（弧度）。

    由 radialGradientRad 轉換而來，會正規化到 -360~360 內再轉弧度。


    startAngle : real

    弧形起始角度（度數）。
