    Detailed Description

    BaseEllipse 是基於 SCDraw2D 模組的橢圓繪製元件， 支援以下功能：

    • 橢圓路徑自動計算
    • 線條與填充漸層
    • 放射性與任意方向漸層
    • 可設定填充起始角度
