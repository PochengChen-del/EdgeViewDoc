    Detailed Description

    BaseShapeEx 元件管理多條 BaseShape 路徑，並提供：

    • 支援任意點的繪製，包括 Bezier 控制點
    • 多條路徑自動生成 BaseShape 物件
    • 支援線性與放射性漸層、填色、虛線、pipe 前後景色
    • 同步或非同步更新各條路徑的屬性

    Property Documentation

    asynchronous : bool

    是否非同步地建立 BaseShape 物件。


    beClose : bool

    是否封閉路徑。


    beFillColor : bool

    是否啟用填色。


    capType : int

    線段端點樣式：0: butt, 1: round。


    dashArray : var

    虛線陣列，最多四個值：[dash0, gap0, dash1, gap1]。


    dashOffset : real

    虛線偏移量。


    dashedImageMirror : bool

    虛線圖案是否鏡像。


    dashedImageSrc : string

    虛線圖案來源。


    fillColor : color

    填色顏色。


    fillDirection : int

    填色方向。 0: bottom → top 1: top → bottom 2: left → right 3: right → left 4: clockwise 5: counterclockwise


    fillGradient : var

    填色漸層定義，格式：[ {position, color}, ... ]。


    fillGradientCenterX : real

    放射性漸層中心 X。


    fillGradientCenterY : real

    放射性漸層中心 Y。


    fillGradientEndX : real

    填色漸層終點 X。


    fillGradientEndY : real

    填色漸層終點 Y。


    fillGradientRadiusX : real

    放射性漸層 X 軸半徑。


    fillGradientRadiusY : real

    放射性漸層 Y 軸半徑。


    fillGradientStartX : real

    填色漸層起點 X。


    fillGradientStartY : real

    填色漸層起點 Y。


    fillRatio : real

    填色比例 (0~1)。


    fillStartRad : real

    填色起始角度（弧度）。


    gradientAnyDir : rect

    任意方向線性漸層資訊 (startX, startY, endX, endY)。


    gradientRadialInfo : rect

    放射性漸層資訊 (centerX, centerY, radiusX, radiusY)。


    gradientType : int

    線性漸層類型： 0: 無漸層 1: 左上 → 右下 2: 左下 → 右上 3: 上 → 下 4: 左 → 右


    joinType : int

    線段連接樣式：0: bevel, 1: round, 2: miter。


    lineColor : color

    描邊顏色。


    lineGradient : var

    描邊漸層定義，格式：[ {position, color}, ... ]。


    lineGradientType : int

    線性漸層類型。


    lineWidth : real

    線寬。


    pathDashOffsets : var

    每條路徑的虛線偏移量陣列。


    pathList : var

    使用者提供的多條路徑資料。

    每條路徑為陣列，每個點格式：

    {
    x: number,
    y: number,
    preCtrl: { x: number, y: number } | null,
    nextCtrl: { x: number, y: number } | null
}

    paths : var

    自動計算生成的多條繪製點陣列。

    由 pathList 計算生成，包含 Bezier 處理，通常不需直接修改。


    pieInfo : rect

    圓弧或扇形資訊，格式: rect(centerX, centerY, startAngle, endAngle)。


    pipeBackColor : color

    管道背景色，僅 pipe 類型使用。


    pipeFrontColor : color

    管道前景色，僅 pipe 類型使用。


    radialGradientRad : real

    放射性漸層旋轉角度（弧度），-360 ~ 360 度。


    rectRadius : real

    矩形角半徑。


    syncConfigToAllPath : bool

    是否同步更新 ShapeEx 的屬性給所有 BaseShape。


    useDash : bool

    是否使用虛線。


    visiblePipeFrontColor : bool

    是否顯示 pipe 前景顏色。
