    Detailed Description

    BasePipeOverlap 是一個複合型的管線繪製元件，基於 BaseShapeEx 與 BasePipePath 實作。

    該元件支援：

    • 雙層管線（背景管線 + 前景流動線）
    • 虛線與虛線動畫（flow）
    • 多路徑獨立流向控制
    • 線條與填色漸層

    常見用途包含 SCADA、流程圖、工業管線顯示與動態流向視覺化。

    Property Documentation

    contentDashArray : var

    前景管線虛線設定陣列。

    格式為：

    [dashLength, gapLength, dashLength2, gapLength2]

    contentDashOffset : real

    前景管線虛線的位移量。


    contentLineColor : color

    前景管線的線條顏色。


    contentLineGradient : var

    前景管線的線條漸層設定。 每個元素包含 position 與 color 欄位。


    contentLineGradientType : int

    前景管線的線條漸層類型。


    contentLineWidth : real

    前景管線的線條寬度。


    contentPathDashOffsets : var

    各路徑獨立的虛線位移量設定。


    contentUseDash : bool

    是否對前景管線啟用虛線。


    dashArray : var

    背景管線的虛線設定陣列。


    dashedImageMirror : bool

    是否對虛線圖片進行鏡像顯示。


    dashedImageSrc : string

    用於虛線繪製的圖片來源路徑。


    flowDirection : bool

    虛線流動方向。 true 表示正向，false 表示反向。


    flowEnable : bool

    是否啟用虛線流動動畫。


    flowStep : int

    每次更新時虛線位移的步進值。


    hasContent : bool

    是否顯示前景管線內容。


    lightColor : color

    前景（高亮）管線顏色。


    lineColor : color

    背景管線顏色。


    pathFlowDirections : var

    各路徑的流動方向設定陣列。


    Method Documentation

    void onDashFlowUpdate()

    更新虛線流動狀態。

    當 flowEnable 為 true 時，會依據 flowStep、 flowDirection 與 pathFlowDirections 更新虛線位移量。
