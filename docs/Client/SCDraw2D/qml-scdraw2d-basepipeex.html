    Detailed Description

    BasePipeEx 是一個對外使用的管線元件，內部會依據 overlapMode 自動切換使用 BasePipe 或 BasePipeOverlap 進行繪製。

    該元件主要用於 SCADA、流程圖、工業管線顯示等情境， 並支援虛線流動、雙層管線、高亮顯示與漸層填色。

    模式說明

    • overlapMode = true：使用 BasePipeOverlap，支援管線重疊與雙層效果
    • overlapMode = false：使用 BasePipe，效能較佳

    內層管線漸層類型。

    內層管線的線條寬度。

    fillColor : color

    每次流動更新的位移步進值。


    gradientType : int

    填色漸層類型。

    線條接合樣式。
