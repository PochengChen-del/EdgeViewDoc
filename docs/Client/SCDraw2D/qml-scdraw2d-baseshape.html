    可繪製任意多點形狀，支援線性與放射性漸層，以及可選 Bezier 曲線。. More...

    Detailed Description

    BaseShape 元件可依 pointList 自動生成繪製點， 並支援：

    • 封閉或開放路徑
    • 每段線段可自動判斷是否使用 Bezier 控制點
    • 線性與放射性漸層顏色

    Property Documentation

    fillGradient : var

    填色使用的線性漸層設定。


    lineGradient : var

    描邊使用的線性漸層設定。

    陣列元素需包含：

    • position：0.0 ~ 1.0
    • color：顏色值

    pointList : var

    形狀的頂點列表。

    陣列元素結構為：

    {
    x: number,
    y: number,
    preCtrl: { x: number, y: number } | null,
    nextCtrl: { x: number, y: number } | null
}

    preCtrl/nextCtrl 用於指定 Bezier 曲線控制點，可選。


    points : var

    自動生成的繪製點座標陣列。

    由 pointList 與 Bezier 控制點計算產生， 使用者通常不需直接修改。


    radialGradientRad : real

    放射性漸層旋轉角度（單位：度）。

    有效範圍 -360 ~ 360。 正值為順時針旋轉。
