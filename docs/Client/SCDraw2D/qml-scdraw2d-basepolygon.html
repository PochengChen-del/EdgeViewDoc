    Detailed Description

    BasePolygon 用於依據 pointList 定義的頂點資料， 自動產生封閉的多邊形路徑。

    支援：

    • 線性漸層（線條 / 填色）
    • 任意方向線性漸層
    • 放射性漸層（含旋轉角度）
