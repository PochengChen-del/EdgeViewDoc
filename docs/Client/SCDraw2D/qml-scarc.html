    Detailed Description

    ScArc 可以用於各種儀表板、進度條或動畫效果。 支援線性與放射性漸層填色，並可自動計算弧形上的點。

    範例用法

    ScArc {
    width: 200
    height: 200
    fillColor: "red"
    lineColor: "blue"
    fillRatio: 0.75
    gradientType: 6
    gradientRadialInfo: Qt.rect(50, 50, 40, 40)
}

    Property Documentation

    beClose : bool

    是否閉合形狀。


    beFillColor : bool

    是否填充顏色。


    dashArray : list<real>

    虛線樣式列表 [線長, 間隔, ...]。


    dashOffset : real

    虛線偏移量。


    fillColor : color

    填充顏色。


    fillDirection : int

    填充方向：

    • 0: bottom to top
    • 1: top to bottom
    • 2: left to right
    • 3: right to left
    • 4: clockwise
    • 5: counterclockwise

    fillRatio : real

    填充比例，範圍 0.0~1.0。


    fillStartRad : real

    填充起始角度 (弧度 0~2\pi)。


    gradientAnyDir : rect

    當 gradientType 為 5 時，指定兩點方向的矩形區域。


    gradientRadialInfo : rect

    當 gradientType 為 6 時，指定徑向漸層資訊 (cx, cy, major_radius, minor_radius)。


    gradientRadialRad : real

    徑向漸層的旋轉角度或半徑參數。


    gradientStop : variant

    漸層停點，格式 [progress, color, ...]。 例如：[0.0, "#FF0000", 1.0, "#0000FF"]


    gradientType : int

    漸層類型：

    • 0: disable
    • 1: left-top to right-bottom
    • 2: left-bottom to right-top
    • 3: top to bottom
    • 4: left to right
    • 5: Any two points (使用 gradientAnyDir)
    • 6: Radial (使用 gradientRadialInfo)
    • 7: Angular

    lineColor : color

    線條顏色。


    lineGradientStop : variant

    線條漸層停點，格式 [progress, color, ...]。


    lineGradientType : int

    線條漸層類型：0: disable, 1: top-to-down, 2: left-to-right。


    lineWidth : real

    線條寬度。


    points : list<real>

    多邊形或弧形的點列表。


    useDash : bool

    是否使用虛線。
