    Detailed Description

    WALabel

    用於顯示靜態或動態文字，支援：

    • 前綴（prefix）、內容（value）、後綴（suffix）組合顯示
    • 字型樣式、對齊方式與換行設定
    • 捲動顯示（ScrollView）
    • 跑馬燈（Marquee）動畫
    • 滑鼠互動暫停跑馬燈

    此元件常用於儀表、狀態顯示與即時資料呈現。

    Property Documentation

    bold : bool

    是否使用粗體字型。


    fontFamily : string

    文字使用的字型家族。


    horizontalAlignment : int

    文字的水平對齊方式。


    italic : bool

    是否使用斜體字型。


    loaderItem : Item

    內部 Label Loader 載入完成後的 Item。


    marquee : bool

    是否啟用跑馬燈顯示模式。


    marqueePause : string

    跑馬燈暫停方式，可為：

    • "none"
    • "hover"
    • "click"

    marqueeSpeed : int

    跑馬燈速度（1～10，數字越大速度越快）。


    pixelSize : int

    文字大小（像素）。


    prefix : string

    顯示文字的前綴內容。


    scrollbar : int

    ScrollBar 顯示策略。


    suffix : string

    顯示文字的後綴內容。


    textColor : color

    文字顏色。


    value : string

    主要顯示的文字內容。


    verticalAlignment : int

    文字的垂直對齊方式。


    wrapMode : int

    文字的換行模式。
