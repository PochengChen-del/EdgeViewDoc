    Detailed Description

    WaObject

    是所有 Widget 的基礎類別，負責：

    • Widget 生命週期管理（建立 / 銷毀）
    • 全域資源註冊與解除註冊
    • Widget 命名與唯一性管理
    • 父子 Widget 關係維護

    Property Documentation

    componentId : string [required]

    Widget 的元件類型識別。


    handleDestroyingManual : bool

    當前 Widget 是否手動處理銷毀流程。


    isDestroying : bool

    指示 Widget 是否正處於銷毀流程中。


    waName : string

    顯示用的 Widget 名稱，需在同一 Drawing 內保持唯一。


    waid : string [required]

    Widget 的唯一識別 ID。


    Signal Documentation

    destroying()

    在 Widget 即將被銷毀時發送。

    Note: The corresponding handler is onDestroying.


    Method Documentation

    string getName()

    回傳目前 Widget 的名稱。


    bool setId(string newId)

    設定 Widget 的唯一 ID。

    \paramnewId 新的 Widget ID。 \return 是否設定成功。
