    Detailed Description

    BaseWidget

    是所有可視化 Widget 的基礎類別，提供：

    • 變形（旋轉、縮放、平移）
    • 動畫與事件觸發機制
    • Drawing 座標與層級計算
    • 編輯模式輔助資訊

    此類別通常不直接使用，而是由具體 Widget 繼承。

    Property Documentation

    angle2Drawing : real [read-only]

    相對於 Drawing 的旋轉角度。


    animations : var

    動畫設定物件。

    animations

    包含 displays、frame、actions、path 等動畫與互動設定， 其結構由系統內部定義，通常由編輯器產生。


    level : int [read-only]

    Widget 在 Drawing 中的層級。


    offset2Drawing : point [read-only]

    相對於 Drawing 的位移。


    type : string [read-only]

    Widget 類型識別字串。


    Signal Documentation

    afterPositionChanged(real diffX, real diffY)

    在 Widget 位置變更完成後發送。

    Note: The corresponding handler is onAfterPositionChanged.


    afterSizeChanged(real diffWidth, real diffHeight)

    在 Widget 尺寸變更完成後發送。

    Note: The corresponding handler is onAfterSizeChanged.
